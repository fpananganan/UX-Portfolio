Travel Resource Network

UX | UI Design | Brand Strategy | Research
Length: 3 Months

Deliverables

User Surveys, Competitive Analysis, User Personas, User Stories, User Flows, Wireframing, Prototyping and Testing

Design Tools

Figma, InVision, Adobe Creative Suite, Xtensio and Lucid Chart

Travel Resource Network is a mobile first application that focuses on disrupting the workflow of nurse related contract work and aims to redefine the process of how these traveling healthcare professionals source and apply to short-term assignments.

What is Travel ResourceNetwork?

A position in travel nursing can be a very lucrative and rewarding career. A travel nurse is typically contracted to work at a specific location for a limited amount of time.

These contracts, otherwise known as assignments, usually last for 13-week periods and can often extend out to 52-week assignments, depending on the client's need.

Travel Resource Network aims to change that. The idea is a unified mobile application that only sources travel assignments for these clinicians.

Problem

Sourcing contract assignments can sometimes be problematic for travel nurses as they are usually navigating in and out of a handful of career websites. Not to mention that the steps in creating accounts and filtering out "non-nursing positions" when looking for contract assignments - a process that can prove to be quite tedious and time consuming.

Solution

Travel Resource Network solves this problem by eliminating the redundancy found in creating multiple user accounts with various career applications.

As a mobile application dedicated to the travel nursing sector that is only meant for travel nurses - introduces a one-stop-shop benefit for users to review locations where assignments are offered, preview pay packages, and obtain any other information relevant to the assignment, all while keeping the primary focus and goal for the end-user in the forefront.
